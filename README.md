
# 📝 JS Quote Generator

A simple and interactive web application that generates random inspirational quotes with a single click. Built using **HTML, CSS, and JavaScript**, this project demonstrates core front-end concepts like DOM manipulation and event handling in a clean and user-friendly interface.

---

## 🚀 Live Demo

🔗 [https://moustafargb-dev.github.io/js-quote-generator/](https://moustafargb-dev.github.io/js-quote-generator/)

---

## ✨ Features

* 🎯 Generate random quotes instantly
* 👤 Display quote author dynamically
* ⚡ Fast and lightweight (no external libraries)
* 📱 Responsive and clean UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
